📝 Fase 1: Análisis de Evidencia Textual

Un aprendiz del programa Tecnólogo en Gestión Logística envió el siguiente ensayo sobre optimización de inventarios. Usa Gemini para evaluar si cumple con los criterios de la competencia.

Evidencia - Ensayo del aprendiz:

"La gestión de inventarios es fundamental para las empresas. Los sistemas de inventario pueden ser perpetuos o periódicos. El método ABC clasifica los productos en tres categorías según su valor. Los productos A son los más importantes y requieren mayor control. Es importante hacer conteos físicos regularmente para evitar faltantes."

📋 Prompt sugerido para copiar:

Actúa como instructor SENA evaluando evidencias de aprendizaje. Analiza el siguiente ensayo de un aprendiz de Gestión Logística sobre inventarios: [PEGA AQUÍ EL TEXTO DEL ENSAYO] Evalúa según estos criterios de la Taxonomía de Bloom (nivel COMPRENDER): 1. ¿Explica claramente los conceptos clave? 2. ¿Relaciona los conceptos entre sí? 3. ¿Incluye ejemplos o aplicaciones? 4. ¿Usa terminología técnica apropiada? Presenta tu análisis en formato de lista con cada criterio y si cumple (SÍ/NO/PARCIAL).

✅ Validación de Fase

¿Gemini identificó correctamente los 4 criterios?

📷 Fase 2: Análisis de Evidencia Visual

Un aprendiz de Soldadura envió una foto de su práctica de cordón de soldadura SMAW. Usa Gemini para analizar la calidad técnica de la imagen.

Evidencia - Imagen de soldadura:

💡 En la práctica real, subirías la foto directamente a Gemini. Para este ejercicio, describe a Gemini que analizarás una foto de cordón de soldadura y pídele que te diga qué aspectos debería evaluar.

📋 Prompt sugerido para copiar:

Actúa como instructor SENA de Soldadura evaluando evidencias prácticas. Voy a subir una imagen de un cordón de soldadura SMAW realizado por un aprendiz. Antes de que la vea, dime: ¿Qué 5 aspectos técnicos clave debo evaluar en la imagen para determinar si la técnica es correcta según normas de calidad en soldadura? Luego, cuando suba la imagen, analízala según esos 5 criterios y dame una calificación de 1-5 para cada uno.

✅ Validación de Fase

¿Gemini te dio criterios técnicos de evaluación?

🎯 Fase 3: Análisis Multimodal Completo

Un aprendiz de ADSI envió su proyecto formativo (PDF) con diagramas de base de datos. Usa Gemini para generar una rúbrica de evaluación completa.

Evidencia - Proyecto formativo:

💡 En la práctica real, subirías el PDF con diagramas. Para este ejercicio, describe el proyecto y pide a Gemini que genere una rúbrica adaptada.

📋 Prompt sugerido para copiar:

Actúa como instructor SENA de Análisis y Desarrollo de Software. Necesito evaluar un proyecto formativo que incluye: - Documento de diseño de base de datos (modelo entidad-relación) - Diagramas UML (casos de uso y clases) - Código SQL para crear las tablas Genera una rúbrica de evaluación con: 1. 4 criterios clave a evaluar 2. 3 niveles de desempeño para cada criterio (Insuficiente, Aceptable, Excelente) 3. Descripción clara de qué cumple cada nivel 4. Puntaje numérico (escala 1-5) Presenta la rúbrica en formato de tabla.

✅ Validación de Fase

¿Gemini generó una rúbrica completa y útil?
